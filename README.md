Muniz - React - TPL

## Tech Stack

react、react-router、mobx、webpack5、less、Web3、Metamask

## Project Structure:

### Enviroment:

1. Install [Node.js](https://nodejs.org/en/download/);
2. Install **yarn**;

```bash
$ npm install -g yarn
```

3. Run：

```bash
$ yarn install
```

### Data init

abi directory：/src/abis/

contract address：/src/constants/tokens.ts

### Development:

```bash
$ yarn dev
```

### Build：

```bash
$ yarn build
```

> assets are in **"/dist"**
